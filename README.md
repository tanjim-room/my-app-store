# NOA IO App Store

## Description
NOA IO App Store is a React-based app marketplace interface where users can browse apps, view app details, install and uninstall apps, and sort installed apps by download count. Installed app state is persisted with localStorage so selections remain available after page reload.

## Technologies
- React 19
- Vite 8
- React Router 7
- Tailwind CSS 4
- DaisyUI 5
- React Icons
- React Toastify
- Recharts
- ESLint
